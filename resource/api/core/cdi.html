cdi¶

API¶

cdi_recon Run reconstruction with difference map algorithm.

Helper methods¶

pi_modulus Transfer sample from real space to q space.
find_support Update sample area based on thresholds.
cal_diff_error Calculate the error in q space.
generate_random_phase_field Initiate random phase.
generate_box_support Generate support area as a box for either 2D or 3D cases.
generate_disk_support Generate support area as a disk for either 2D or 3D cases.
